Formats des fichiers d'importation

Instructions générales
  • Tous les fichiers doivent être au format CSV
  • Les dates doivent être au format DD/MM/YYYY
  • Les nombres décimaux doivent utiliser le point comme séparateur
  • Les champs sont séparés par des virgules
  • La première ligne doit contenir les en-têtes
Format Employés

Exemple :
1,Rakoto,Alain,Masculin,03/04/2024,01/01/1980,My Company
Format Structure Salariale

Exemple :
STR001,Base,BS,E,B,My Company
Format Attribution Salaires

Exemple :
01/01/2024,1,500000,STR001
